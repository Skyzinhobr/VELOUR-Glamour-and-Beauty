# VELOUR---Glamour-and-Beauty
Projeto desenvolvido para a disciplina de Programação Web, simulando um e-commerce de cosméticos com HTML, CSS e JavaScript. Possui catálogo dinâmico, filtros de produtos, wishlist, carrinho de compras interativo, animações modernas e layout responsivo, aplicando conceitos de manipulação do DOM e experiência do usuário. ✨💄💻
